# 多角度 3D 编辑器项目文档

## 项目概述

**angle-editor-3d** 是一个基于 Vue 3 和 Three.js 的 3D 角度编辑器应用。该项目提供了一个交互式的 3D 编辑界面，用户可以通过两种不同的模式来调整和预览图像或 3D 对象的角度。

### 主要功能

- **双模式编辑器**
    - **主体模式（Subject）**：允许用户直接旋转和调整主体对象的角度
    - **摄像头模式（Camera）**：通过调整摄像头的位置来改变观察角度

- **交互式控制**
    - 旋转控制：水平方向旋转
    - 倾斜控制：垂直方向倾斜
    - 缩放控制：三个固定档位（广角、中等、特写）

- **交互方式**
    - 鼠标拖拽支持
    - 触摸屏支持
    - 滑块控制
    - 方向按钮（摄像头模式）

- **视觉反馈**
    - 实时 3D 渲染
    - 球形网格辅助线
    - 3D 摄像头模型
    - 平滑的动画过渡

### 技术栈

- **前端框架**: Vue 3.4.21 (Composition API)
- **3D 引擎**: Three.js 0.183.2
- **构建工具**: Vite 5.2.0
- **开发语言**: TypeScript 5.4.2

### 核心组件说明

#### AngleEditor.vue
主编辑器组件，负责：
- 模式切换（主体/摄像头）
- 状态管理（旋转、倾斜、缩放）
- 用户交互处理
- UI 控制面板

#### ThreeScene.vue
Three.js 3D 场景组件，负责：
- 初始化 Three.js 场景、相机、渲染器
- 创建和管理 3D 对象（立方体、球体网格、摄像头模型）
- 处理拖拽交互
- 更新 3D 变换

#### Slider.vue
自定义滑块组件，提供：
- 平滑的拖拽体验
- 触摸支持
- 可配置的范围和值

## 开发约定

### 代码风格

- 使用 Vue 3 Composition API (`<script setup>`)
- 使用 TypeScript 进行类型定义
- 优先使用计算属性（`computed`）处理派生状态
- 使用 `watch` 监听响应式数据变化
- 组件职责单一，保持简洁

### 命名规范

- 组件文件：PascalCase（如 `AngleEditor.vue`）
- 变量/函数：camelCase（如 `subjectRotation`）
- 常量：UPPER_SNAKE_CASE（如 `MAX_VALUE`）
- CSS 类：kebab-case（如 `angle-editor`）

### 注释规范

- 所有函数添加中文注释，说明功能
- 复杂逻辑添加行内注释
- 保持注释简洁明了

### 性能优化

- 使用 `requestAnimationFrame` 进行动画渲染
- 适时清理事件监听器和 Three.js 资源
- 避免不必要的重渲染

## 功能说明

### 主体模式（Subject Mode）

- **默认状态**：旋转 45°，倾斜 -30°，缩放中等（1.15）
- **交互方式**：直接拖拽场景或使用滑块控制
- **缩放范围**：
    - 特写：1.5
    - 中等：1.15
    - 广角：0.9

### 摄像头模式（Camera Mode）

- **默认状态**：旋转 -45°，倾斜 -30°，缩放中等（1.15）
- **交互方式**：方向按钮或拖拽场景
- **缩放范围**：
    - 特写：1.5
    - 中等：1.15
    - 广角：0.9
- **辅助元素**：球形网格（包含赤道线）、3D 摄像头模型
- **正方体位置**：模式切换时，主体模式为 (0, 0, 0)，摄像头模式为 (0, 0, 1.7)

### 缩放控制

缩放采用三档固定模式，拖动时自动吸附到最近的档位：
- 0-33：特写
- 34-66：中等
- 67-100：广角

### 重置功能

- **主体模式重置**：旋转 45°，倾斜 -30°，缩放中等
- **摄像头模式重置**：旋转 -45°，倾斜 -30°，缩放中等

## 3D 场景参数

### 立方体
- **尺寸**：1.3
- **圆角半径**：0.1
- **材质**：前面用于贴图，其他面显示文字标签

### 球体网格
- **半径**：1.6
- **经线间隔**：9°
- **纬线间隔**：18°
- **关键线条**：-90°, -45°, 0°, 45°, 90°（更深的颜色）

### 摄像头模型
- **主体尺寸**：宽 0.4，高 0.28，深 0.15
- **圆角半径**：0.04
- **球面半径**：1.2

## 浏览器兼容性

- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 支持触摸屏设备
- 需要支持 WebGL 的浏览器

## 开发注意事项

1. **Three.js 资源管理**：组件卸载时必须清理 WebGL 资源
2. **事件监听**：及时移除不再需要的事件监听器
3. **响应式更新**：Three.js 变换需要手动触发更新
4. **触摸事件**：同时支持鼠标和触摸事件
5. **状态同步**：确保 UI 状态与 3D 场景状态保持同步
6. **模式切换**：切换模式时重置缩放和位置参数
